官方说明：

[ 网络 ]

  • 由于安全原因，dhcpcd默认情况下未配置，或者使用’systemctl start dhcpcd’手动启动，或’systemctl enable dhcpcd’开机自启动.

[ 安装 ]

  • 安装BlackArch Linux，只需运行’blackarch-install’并按照官方说明安装.

[ EXPLOITS ]

  • 升级exploit archives在/usr/share/exploits使用’sploitctl’.(Github sploitctl)

[ MISC ]

  • ChangeLog可以在ISO的/root中找到.

初始化

初始用户名：root，密码：blackarch

查看网卡
ip addr

开启网卡
ifconfig 网卡 up

开启动态获取IP
dhcpcd

编辑安装文件，修改ping的网站，默认通过ping github来检测是否有网络

修改国内源

安装BlackArch

之后
blackarch-install

询问从什么位置安装linux
2.Install from BlackArch Live-ISO

设置键盘布局
1

默认输入
us

设置主机名，可随意，到时候可以在/etc/hostname里面修改
TroAZ

之后设置网卡，输入
ens33

选择网络环境，动态ip
1. Auto DHCP (use this for auto……

之后会询问是否检测全球最新的源，输入
n

检测硬盘，输入
sda

是否用cfdisk创建分区，输入
y

是否清空内存，输入
y

选硬盘模式，选择
dos

点new创建新分区（BlackArch虚拟机我默认给了60G的磁盘）
1G

再点primary

再点BootableB

之后方向键 “↓” 键

继续点new创建新分区
1G

再点primary

再选type

选linux swap solaris

再按方向键 “↓” 键

继续点new创建新分区
58G

再点primary

再选write，输入
yes

选quit离开

询问你是否加密根分区，输入
y

之后输入分区类型

然后是否确认分区，输入
y

提示格式化分区，输入
y

之后设置分区密码

再输入分区密码，确认是否可行

再设置root密码

建立一个普通用户

设置时区
Asia/Shanghai

再选择mirrot国内源

其他

BlackArch开机后默认没有网络：

关于vi，数字键盘无法输入数字，把用户环境的vi=vim屏蔽去掉，再应用用户环境变量

source ~/.bashrc # 应用用户环境变量

抓包工具wireshark错误：
vi /usr/share/wireshark/init.lua

将倒数第二行：
dofile(DATA_DIR.."console.lua")
改为
--dofile(DATA_DIR.."console.lua")
